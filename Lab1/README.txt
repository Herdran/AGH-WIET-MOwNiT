Jak skompilować?

Do uruchomienia programu potrzebny jest Python w wersji 3.0 lub wyższej, a także biblioteka numpy oraz matplotlib, które można zainstalować wpisując w wierszu poleceń: pip install numpy, pip install matplotlib.

Co zwraca program?

Program powinien wypisywać tabelę wszystkich wartości xk dla k=[0,34] oraz osobno dla k=34 trzykrotnie, dla trzech typów zmiennych. Następnie powinien narysować trzy wykresy zależności wartości x od k

wartości wyrazów o numerach 1, 5, 10, 15, 20, 25 oraz 30 dla trzech typów danych obliczanych na dwa sposoby (związkiem podanym w zadaniu oraz związkiem zmodyfikowanym). Dla pierwszego sposobu program powinien wyrzucić trzy błędy, sygnalizujące próbę dzielenia przez liczbę o bardzo małej wartości (~=0).
